# Calcifying_plankton_paper

This repository contains all data files and R code used for the analysis, related to the manuscript "The contributions of various calcifying plankton to the South Atlantic calcium carbonate stock"

For questions about the data and code in the repository, please contact the author Anne Kruijt, annelaurakruijt@hotmail.com

Data files:
Each excel file contains an info tab, explaining the nature of the data

Code:
All scripts run in R or Rstudio. If the repository is cloned, all scripts should directly run from within the folder, using the data files.
